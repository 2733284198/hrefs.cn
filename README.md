本仓库是爱斯园网站后台管理项目，它是一个纯 vue 项目，接口依赖仓库（https://github.com/iissy/goweb），如需开发调试，需要启动两个项目。

## asy 2.0 Released 2020年02月23日
+ 前端路由支持 vue-router
+ 富文本编辑器 ckeditor4-vue
+ axios 引入
+ 支持跨域接口调用

## 包安装
